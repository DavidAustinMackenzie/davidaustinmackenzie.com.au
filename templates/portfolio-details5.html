Full stack web app with Angular Frontend and Django Backend.

This was a web app that I created for a client that wanted to create unique randomly generated numbers for the Xmas Work Party Bad Santa Draw. I added some basic client side validation to check for duplicate names, the draw is not selected and also to ensure that unique random numbers are generated. This was a fun introduction to working with Angular and Django and I learnt lots about both frameworks. I deployed this app to 2 servers a Unix server for the BackEnd and a Linux server for the FrontEnd on AWS for the client to access.

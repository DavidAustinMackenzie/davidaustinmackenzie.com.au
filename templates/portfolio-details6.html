Full stack web app with ASP.NET MVC Core using Razor pages for displaying Front end.

This full stack web app is in development. It uses ASP.NET MVC core for the application that is connected to a SQL Server to store the data. Using Entity framework to map all the entities, I have used the built in scaffolding to create all the required entities. I have modified the logic for the backend and also the front end to meet the requirements of the client. The client requires a way to manage all the bookings for meeting rooms.
